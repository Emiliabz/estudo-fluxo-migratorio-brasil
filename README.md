# 📊 Estudo de Fluxo Migratório: Brasil e Argentina para o Canadá (1980 - 2013)

Este projeto apresenta uma análise comparativa da imigração de cidadãos do Brasil e da Argentina para o Canadá ao longo de três décadas. O objetivo principal foi explorar e estruturar os dados históricos para construir uma visualização temporal dinâmica e interativa.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Python**: Linguagem principal para manipulação e lógica do script.
* **Pandas**: Limpeza, estruturação e tratamento dos dados temporais.
* **Plotly**: Construção de gráficos interativos e criação de frames para animação em tempo real.
* **GitHub**: Controle de versão e hospedagem do portfólio.

## 📈 O que foi desenvolvido neste estudo?
* **Tratamento de Dados**: Conversão e limpeza de variáveis temporais (garantindo que os anos fossem tratados de forma estritamente numérica para evitar quebras de renderização).
* **Análise Comparativa**: Mapeamento do volume de imigrantes ano a ano entre os dois países da América do Sul.
* **Gráfico Animado Interativo**: Implementação de uma linha do tempo dinâmica utilizando o recurso de animação do Plotly, permitindo visualizar a evolução dos fluxos ao clicar no botão **Play**.

## 📂 Estrutura do Repositório
* `analise_imigracao.py`: Código-fonte em Python com o tratamento de dados e lógica da animação.
* `imigracao_america_sul_interativo.ipynb`: Notebook do Google Colab onde os gráficos e tabelas podem ser vistos direto na página.
* `imigrantes_canada.csv`: Base de dados original utilizada no estudo.
* `imigracao_america_sul_interativo.html`: Arquivo do gráfico interativo para download e visualização direta no navegador.

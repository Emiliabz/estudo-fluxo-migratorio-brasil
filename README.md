# 📊 Estudo de Fluxo Migratório para o Canadá (1980 - 2013): Foco na América do Sul

Este projeto apresenta uma análise detalhada e dinâmica sobre a imigração de cidadãos de países da América do Sul para o Canadá ao longo de três décadas. O estudo evolui de uma perspectiva geral focada no Brasil para análises agrupadas dos maiores fluxos migratórios do continente.

## 📈 Cenários Analisados no Estudo
O projeto foi estruturado para gerar e comparar três visões estratégicas de dados:
1. **Fluxo Individualizado**: Análise direcionada ao volume de imigrantes vindos do Brasil.
2. **Top 4 da América do Sul**: Identificação e evolução temporal dos 4 países do continente com maior volume de imigração.
3. **Top 10 da América do Sul**: Visão macro comparativa englobando os 10 principais fluxos migratórios sul-americanos para o território canadense.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Python**: Linguagem principal para estruturação lógica do projeto.
* **Pandas**: Filtragem de dados, ordenação por ranking (Top 4 e Top 10) e tratamento de séries temporais.
* **Plotly**: Construção de gráficos dinâmicos e animações temporais interativas com controle de reprodução (botão Play).
* **GitHub**: Hospedagem do portfólio e controle de versão.

## 📂 Estrutura do Repositório
* `analise_imigracao.py`: Script em Python com a lógica de filtros, agrupamentos do ranking e criação das animações.
* `imigracao_america_sul_interativo.ipynb`: Notebook do Google Colab com todas as tabelas e códigos gerados.
* `imigrantes_canada.csv`: Base de dados original utilizada no estudo.
